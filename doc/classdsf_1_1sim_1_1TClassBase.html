Detailed Description

template<class BClass>
class dsf::sim::TClassBase< BClass >

TClassBase holds the base class, B, of derivative class DClass. TClass inherits TClassBase an stores DClass. This inheritance allows TClasses containing varied classes to exist in the same dictionary if they share base class B
