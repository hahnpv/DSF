Detailed Description

A digital clock. All core time calculations are done with integers, resulting in no round-off errors that must be corrected as the simulation proceeds. The only time a double appears is in the return values, by type conversion with an error function. The error function is 2/dt. This is because the clock updates twice per integration cycle.

bool dsf::sim::Clock::Sample ( double  t)
inline

A dual-use sample function. If t=0, it returns true if the sim is not mid-integration, and false if the simulation is currently in the middle of an integration cycle.
If t != 0, then it checks if the current sim time is a multiple of the requested time, and returns true if so.
